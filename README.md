## ðŸ“Œ Gambaran Proyek
Proyek ini bertujuan untuk memprediksi harga jual rumah berdasarkan berbagai fitur properti (seperti luas tanah, kualitas material, tahun pembangunan, dll). Proyek ini dibangun menggunakan dataset **"House Prices - Advanced Regression Techniques"** dari Kaggle.

Solusi ini mencakup seluruh pipeline Machine Learning:
1. **Exploratory Data Analysis (EDA)** untuk memahami distribusi data.
2. **Preprocessing & Feature Engineering** untuk membersihkan data.
3. **Model Training** menggunakan algoritma *Random Forest Regressor*.
4. **Deployment** model menjadi aplikasi web interaktif menggunakan Streamlit.
